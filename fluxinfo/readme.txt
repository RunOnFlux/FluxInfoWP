Provides Flux specific information for WordPress app, also monitors app expiration

== Description ==

Displays dApp expiration date amd other Fulx network info

## Getting Started ##

Install and visit Settings page

## Is it free to use? ##

Yes, this plugin is completely free with no paid version.

Please consider helping out by <a href="https://wordpress.org/support/plugin/info-for-flux/reviews/#new-post">leaving a review</a>, or <a href="https://www.paypal.com/donate/?hosted_button_id=RX28BBH7L5XDS">donate</a>.

== Installation ==

1. Upload 'fluxinfo' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customise plugin settings in WordPress admin at Settings > FluxInfo
