Provides Flux specific information for WordPress app, also monitors app expiration

== Description ==

Displays dApp expiration date amd other Fulx network info

## Getting Started ##

Install and visit Settings page

## Is it free to use? ##

Yes, this plugin is completely free with no paid version.

Please consider helping out by <a href="https://wordpress.org/support/plugin/infoforflux/reviews/#new-post">leaving a review</a>, or <a href="https://www.paypal.com/donate/?hosted_button_id=RX28BBH7L5XDS">donate</a>.

== Third Party Sites ===
This plugin will access API Services (https://api.runonflux.io) of the Flux Network (see https://docs.runonflux.io)
This plugin will also access by IP address Flux Nodes that your application is running in to check the state of the app.

The Flux Application name ('wordpress178.....') is used in a few of the API calls GET only, no data is POSTED.

== Installation ==

1. Upload 'infoforflux.zip' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customise plugin settings in WordPress admin at Settings > infoforflux
